# Coal Industry Forum · Tajikistan 2026 — CMS

Премиум-сайт Международного форума угольной промышленности с полнофункциональной админ-панелью.

## Особенности

- **3 языка**: английский (основной), русский, таджикский
- **Премиум-дизайн**: тёмная угольно-золотая палитра, премиум-типографика (Playfair Display + Inter)
- **Адаптивность**: мобильная, планшетная и десктопная версии
- **CMS**: управление любым контентом через интерфейс на русском
- **Регистрация**: премиальная форма с верификацией email, обязательной загрузкой фото 3×4 и копии паспорта
- **Архитектура «по годам»**: каждый год форума получает свою страницу с галереей фото/видео
- **Новостной блок**: публикация новостей с обложками и переводами
- **SEO**: sitemap.xml с hreflang, Open Graph, JSON-LD Event schema
- **Безопасность**: bcrypt, CSRF-защита, защита от брутфорса, .htaccess

## Технологии

- PHP 8.0+ (тестировано на 8.4)
- MySQL 5.7+ / MariaDB 10+
- Никаких внешних зависимостей (без Composer, без Node.js)
- Совместимо с любым shared-хостингом (Timeweb, Beget, FirstVDS, и т.д.)

## Установка на Timeweb

### 1. Создание БД

В панели Timeweb → **Базы данных MySQL** → **Создать базу**:
- Имя БД: `coalforum` (или любое)
- Пользователь: создайте нового
- Сохраните: имя БД, пользователя и пароль

### 2. Загрузка файлов

Через FTP или файловый менеджер Timeweb загрузите содержимое архива в корневую папку сайта (`public_html`).

**Важно**: содержимое папки `public/` должно лежать в корне (`public_html/`), а остальные папки (`admin/`, `includes/`, `uploads/`, `database/`) — на том же уровне. Структура должна быть такой:

```
public_html/
├── index.php          ← из public/
├── sitemap.php
├── robots.txt
├── .htaccess
├── pages/
├── handlers/
├── layout/
├── assets/
└── admin/             ← рядом
    ├── index.php
    └── ...
public_html/../        ← или один уровень выше, если хостинг позволяет
├── includes/
├── uploads/
├── database/
└── logs/
```

**Самый простой вариант** для Timeweb: положите всё одной папкой и в `public_html/` сделайте симлинк на `public/`, либо просто положите `public/` как `public_html/`.

### 3. Конфигурация

Скопируйте `includes/config.sample.php` → `includes/config.php` и заполните:

```php
'db' => [
    'host' => 'localhost',
    'name' => 'cu12345_coalfor',   // ← из Timeweb
    'user' => 'cu12345_coalfor',
    'pass' => 'ПарольБДизTimeweb',
],
'site_url' => 'https://coalindustry.tj',
'mail' => [
    'host' => 'smtp.timeweb.ru',
    'port' => 465,
    'encryption' => 'ssl',
    'username' => 'no-reply@coalindustry.tj',
    'password' => 'ПарольПочты',
    ...
],
```

### 4. Запуск установщика

Откройте в браузере: `https://coalindustry.tj/install.php`

Заполните форму:
- Логин администратора
- ФИО
- Email
- Пароль (минимум 8 символов)

После установки **удалите файл `install.php`** с сервера.

### 5. Вход в админку

Откройте: `https://coalindustry.tj/admin/`

Войдите с созданными учётными данными.

### 6. Настройка SMTP в админке (если не указали в config.php)

В админке → **Настройки сайта** → секция **SMTP**:
- Сервер: `smtp.timeweb.ru`
- Порт: `465`
- Шифрование: `SSL`
- Логин: `no-reply@coalindustry.tj`
- Пароль: пароль от почты no-reply

В Timeweb почтовые ящики `info@`, `reg@`, `no-reply@coalindustry.tj` создаются в разделе **Почта**.

### 7. Тестирование

- **Главная**: `https://coalindustry.tj/`
- **Регистрация**: `https://coalindustry.tj/?p=register`
- **Программа**: `https://coalindustry.tj/?p=program`
- **Архив**: `https://coalindustry.tj/?p=archive`

## Управление контентом

### Структура «по годам»

Главное преимущество CMS — структура по годам. Каждый год форума — отдельная запись с:
- Темами форума
- Программой
- Спикерами
- Партнёрами
- Документами (программа/концепт-нота PDF на 3 языках)
- Фото- и видеогалереей

### Что менять регулярно

1. **Новости** — раздел «📰 Новости»: новые публикации еженедельно
2. **Спикеры** — по мере подтверждения участия
3. **Партнёры** — по мере подписания соглашений
4. **Регистрации** — мониторинг заявок, экспорт в CSV

### Что менять для нового года

1. **Годы форума** — создать новую запись (например, 2027), отметить «Текущий»
2. **Программа** — добавить пункты программы для нового года
3. **Темы** — продублировать или создать новые темы
4. **Документы** — загрузить новые PDF
5. **Спикеры** — добавить новый список

Прошлые годы автоматически останутся в «Архиве форумов».

## Безопасность

- Все формы защищены CSRF-токенами
- Пароли админов хешируются bcrypt с cost=12
- Защита от брутфорса: после 5 неудачных попыток аккаунт блокируется на 10 минут
- Файлы регистраций (паспорта) защищены от прямого доступа через .htaccess
- Загрузки проверяются по MIME-типу
- В `/uploads/` запрещено выполнение PHP

## Резервное копирование

Регулярно экспортируйте БД через панель Timeweb (раздел «Базы данных» → экспорт SQL).
Также сохраняйте папку `/uploads/` — там все загруженные файлы.

## Учётные данные SMTP-почт

Создайте в панели Timeweb (раздел «Почта»):
- `info@coalindustry.tj` — общие вопросы
- `reg@coalindustry.tj` — уведомления о регистрациях
- `no-reply@coalindustry.tj` — рассылка от системы
